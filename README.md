### 대한민국 헌법 작성 프로젝트

이 프로젝트는 코딩 프로젝트는 아닙니다. 오히려, 문학적 작성 프로젝트에 가깝습니다.  
프로젝트의 명칭은 대한민국 헌법을 보고, 느끼고, 직접 타이핑하는 프로젝트 (See It, Feel It, Type It: The Korean Constitution Project) 입니다.  

진행 기간 : 총 4일  
결과물 : 대한민국 헌법.docx, 일자별 pdf  

프로젝트를 진행한 이유는 단순합니다. 법 중의 법이라고 불리는 헌법은, 국가의 중대사를 관할하며, 모든 법의 기본이 되는 법입니다.  

그 위상에 걸맞게 헌법은 매우 높은 퀄리티로 작성되어있고, 고유의 논리적 흐름을 보유하고 있습니다. 이에, 이 헌법을 작성한 분들의 사고 흐름을 조금이나마 따라가고자 진행하게 되었습니다.  

그런 만큼, 부족하지만 그 분들이 의도하고자 하는 바를 제가 이해했다면 좋겠습니다.  
세상의 지식은 넓은 바다와 같다고 느껴, 이해할 수 없는 것도 많으나 조금이나마 이해하고 싶다고 다가가면 마음을 열지 않을 세계는 없다고 생각합니다.  

감사합니다.
